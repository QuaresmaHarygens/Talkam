# Talkam Liberia Branding Package

## Name Options & Rationale
1. **Talkam Liberia** – colloquial "talk it"; easy recall, community vibe.
2. **Voicesafe Liberia** – highlights safety for whistleblowers.
3. **Kukaton Monitor** – "Kukaton" slang for exposing wrongdoing; investigative tone.
4. **PalavaWatch** – resonates with "palava" (issue/conflict); implies vigilance.
5. **SafeSabi** – mixes "safe" + Liberian "sabi" (to know); knowledge for safety.
6. **LionHeart Signal** – national symbol + courage; appeals to responders.

## Logo System
- **Primary mark:** speech bubble merged with Liberia outline and a shield.
- **Secondary mark:** circular badge with radio waves for app icon.
- **Usage:** solid color for small sizes, gradient for hero surfaces.

## Color Palette
| Name | Hex | Usage |
| --- | --- | --- |
| Sunrise Amber | `#f59e0b` | Primary CTA, highlight states |
| Deep Lagoon | `#0f172a` | Headlines, iconography |
| Kpelle Green | `#16a34a` | Success, verified badges |
| Mano River Blue | `#2563eb` | Links, secondary CTAs |
| Clay Terracotta | `#b45309` | Alerts, urgent reports |
| Cotton Grey | `#f1f5f9` | Background, cards |
| Palm Trunk | `#4b5563` | Body text |

## Typography
- **Display:** `DM Sans` – friendly, legible in small screens.
- **Body:** `Inter` – supports extended Latin, works on Android/iOS/web.
- **Fallbacks:** `system-ui`, `Roboto`, `SF Pro`.

## Iconography Guidelines
- Stroke-based 2px icons, rounded corners, consistent viewbox 24px.
- Icons for categories (Health, Security, Infrastructure, Governance, Environment, Civil Rights).
- Offline-specific icons (queue, sync). Provide outlines with optional filled style for urgent states.

## Tone of Voice
- **Community-oriented:** inclusive, uses Liberian English phrases sparingly.
- **Reassuring:** highlights anonymity, safety features.
- **Actionable:** instructive CTAs, avoids technical jargon.

## Sample Taglines
- "Speak small, change big."
- "Safe reports, fast response."
- "Liberian voices, verified."

## App Icon Concepts
1. **Shield Bubble:** two-tone speech bubble with shield center; gradient from Sunrise Amber to Mano River Blue.
2. **Liberia Outline Pulse:** solid Liberia map silhouette with pulsing rings.
3. **Radio Palm:** abstract palm leaf with radio waves representing rural reach.

## Accessibility & Localization Notes
- All palette colors meet WCAG AA for text/background combos.
- Provide bilingual copy modules; fallback font includes diacritics for Kpelle, Vai.
- Icon ALT tags localized via i18n files.

## Usage Scenarios
- **Mobile:** gradient hero, muted cards.
- **NGO Dashboard:** light mode default, optional dark mode using palette inversions.
- **Print / PDF:** grayscale-friendly via monotone palette derived from Deep Lagoon.
