# Admin Dashboard Setup Guide

## Quick Start

### 1. Access the Dashboard

- URL: https://admin.talkam.liberia.com
- Use your NGO/Government credentials

### 2. First Login

1. Enter phone number and password
2. Dashboard loads automatically
3. Review analytics and reports

## Dashboard Overview

### Analytics Tab

**Key Metrics**:
- Total Reports: All submissions
- Verified Reports: Confirmed by organizations
- Verification Rate: % of verified reports
- Avg Response Time: Time to verification
- Active Alerts: Current alerts

**Charts**:
- Reports by County (bar chart)
- Category Trends (pie chart)
- Time-series trends (coming soon)

### Reports Tab

**Filters**:
- County: Filter by location
- Category: Infrastructure, Security, etc.
- Status: Submitted, Verified, Rejected
- Text Search: Search summaries

**Actions**:
- **Confirm**: Verify report as accurate
- **Reject**: Mark as false/invalid
- **View Details**: Full report information

## Managing Reports

### Verifying Reports

1. Navigate to Reports tab
2. Review report details
3. Click "Confirm" if verified
4. Or "Reject" if invalid
5. Add optional comment

### Broadcasting Alerts

1. Click "Broadcast Alert" button
2. Enter message
3. Select severity level
4. Choose target counties (optional)
5. Select channels (SMS, Push, Both)
6. Send alert

## User Management

### Adding Users

1. Go to Settings → Users
2. Click "Add User"
3. Enter phone number and role
4. User receives SMS with password

### Roles

- **Admin**: Full access
- **NGO**: Can verify reports, view analytics
- **Government**: Can verify, broadcast alerts
- **Moderator**: Can verify, flag reports

## Best Practices

### Verification

- Verify within 24 hours
- Check multiple sources when possible
- Add helpful comments
- Reject false reports promptly

### Alerts

- Use high severity for emergencies only
- Keep messages concise
- Target specific counties when possible
- Test SMS delivery first

### Analytics

- Review dashboard daily
- Track trends by county/category
- Identify high-priority areas
- Export data for reports (coming soon)

## Troubleshooting

### Cannot Login

- Check credentials
- Contact system administrator
- Reset password via SMS

### Reports Not Loading

- Check internet connection
- Refresh page
- Clear browser cache
- Contact support

### Slow Performance

- Check API status
- Reduce filter scope
- Wait for data to load
